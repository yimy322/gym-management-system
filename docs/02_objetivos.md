# Objetivos del proyecto
## Objetivo General
Desarrollar un Sistema Web Integral en beneficio de la Gestión Administrativa del Gimnasio STRKE, el cual permitá optimizar el registro de usuarios, la gestión de membresías, el control de pagos y el seguimiento de asistencia, mejorando la eficiencia operativa y la organización de la información mediante el uso de tecnologías web modernas.

## Objetivos Específicos
1. Analizar los requerimientos funcionales y no funcionales del sistema para identificar las necesidades administrativas del gimnasio STRKE y definir el alcance del proyecto.

2. Diseñar la arquitectura del sistema y la estructura de la base de datos utilizando Spring Boot, Thymeleaf, Bootstrap y MySQL, garantizando una organización eficiente de la información.

3. Implementar los módulos principales del sistema, incluyendo gestión de usuarios, control de membresías, registro de pagos y seguimiento de asistencia, asegurando su correcto funcionamiento.

4. Realizar pruebas funcionales en cada módulo desarrollado para verificar el cumplimiento de los requerimientos establecidos y corregir posibles errores durante el desarrollo.

5. Elaborar la documentación técnica y funcional del proyecto, incluyendo justificación, objetivos, requerimientos y control de versiones mediante GitHub, asegurando la trazabilidad y organización del trabajo colaborativo.
